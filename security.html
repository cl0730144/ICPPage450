Web Safety Tips

Web Apps can access a lot of information from your web browser such as:

Application Name - Name of the web browser displaying the page

Application Version - Version of the web browser displaying the page

Geolocation - API used to access the user's current location and permissions

Screen Height - Display screen's height

Screen Width - Display screen's width

Pixel Depth - Display screen's color resolution in bits per pixel

Access to any of the information above can lead to someone else knowing:

what application you are using, to hack it.

what app version you are using, to hack it.

your physical location, to rob you.

your current device (e.g. phone), to hack it.

your current device again (e.g. tablet), to hack it.

your current device yet again (e.g. laptop), to hack it.
